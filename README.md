# 🧮 My 1st Calculator

A simple and clean calculator application built with Visual Basic .NET in Visual Studio.

## ✨ Features
- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗
- Clean and easy to use interface

## 🛠️ Built With
- Visual Basic .NET
- Visual Studio

## 🚀 How to Run
1. Download the Calculator.exe from the Releases section
2. Double click to run — no installation needed!
